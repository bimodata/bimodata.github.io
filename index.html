Bimo

Une plateforme Open Source de transformation de données de transport public.

Bienvenue !

Modulaire

Bimo est un assemblage de modules pouvant être utilisés ensemble ou séparément selon vos besoins.

Extensible

Grâce à une architecture basée sur l'inversion de contrôle, écrivez des fonctions personnalisées et intégrez les facilement dans des traitements de données pré-existants.

Isomorphe

Tous les modules métiers sont isomorphes: vous pouvez les intégrer côté client sur une page web, dans un traitement complexe sur un serveur en Node.js® ou dans une appli Electron.
